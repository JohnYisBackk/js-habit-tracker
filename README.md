# Habit Tracker ✅

Vanilla JavaScript habit tracker with localStorage.

## Features
- Add new habits
- Mark habits as done
- Delete single habit
- Delete all habits
- Daily progress counter
- Data persistence using localStorage
- Custom scrollbar and clean UI

## Built with
- HTML
- CSS
- Vanilla JavaScript

## What I learned
- JavaScript state management
- DOM manipulation and rendering
- Event handling
- Working with localStorage
- Basic UX decisions (confirm dialogs, empty states)

## Project purpose
This is a learning project focused on building a complete small application
from scratch without frameworks.

## Possible improvements
- Auto reset habits based on date
- Dark mode
- Edit habit names
- Statistics / streak tracking
